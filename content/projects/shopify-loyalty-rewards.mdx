# Shopify Loyalty Rewards

A serverless loyalty system optimized for high-volume merchants, emphasizing cost efficiency and scalability.

## Strategic Impact

- **0% Database Overhead**: Leverages Shopify Metafields for all persistent data, eliminating external infrastructure costs.
- **Automated Tier Logic**: Event-driven order processing via webhooks ensures real-time customer status updates.
- **Spend Velocity**: Implemented custom rolling 12-month spend calculations to drive repeat purchase behavior.
