# OpenClaw

OpenClaw is a production-grade harness for LLM agents, designed to bridge the gap between experimental scripts and reliable production systems.

## Key Features

- **Robust Tool Execution**: Secure sandboxing for agent-called tools.
- **Observability**: Built-in OpenTelemetry integration for tracing agent decisions.
- **State Management**: Durable execution for long-running agent tasks.
- **Provider Agnostic**: Works with OpenAI, Anthropic, and local models via Ollama.

## Why OpenClaw?

In the current landscape, moving from a successful prompt to a production system is fraught with challenges. OpenClaw provides the necessary scaffolding to ensure that your agents are reliable, observable, and secure.
