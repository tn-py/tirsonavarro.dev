# Selector Sage

An agentic recommendation engine that guides users through complex hardware selection flows using LLM-powered reasoning.

## Technical Highlights

- **Guided Logic**: Utilizes Agentic RAG to retrieve technical hardware specs based on natural language intent.
- **Dynamic Reasoning**: Employs LangGraph to manage the conversational state and refine product suggestions iteratively.
- **Integration**: Seamlessly pulls data from Shopify and SearchSpring to provide real-time availability.
