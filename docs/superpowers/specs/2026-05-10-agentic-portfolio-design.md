# Spec: Agentic Architect Portfolio

**Date:** 2026-05-10
**Author:** Gemini CLI
**Subject:** Developer Portfolio for Tirso Navarro

## 1. Vision & Goals
The portfolio serves as a professional showcase for **Tirso Navarro**, an "Agentic Architect" specializing in the intersection of AI Agent infrastructure and high-performance e-commerce operations.

### Primary Objectives
- **Consulting/Freelance:** Demonstrate production-grade reliability for clients seeking AI automation and Shopify optimization.
- **Personal Branding/Community:** Establish technical authority in the AI Agent and Model Context Protocol (MCP) ecosystems.
- **Hybrid Positioning:** Bridge the gap between "experimental researcher" and "production engineer" by highlighting agent harnesses (OpenClaw, Hermes) used in real-world scenarios (UHS Hardware).

---

## 2. Visual Language: "The Agentic Look"
A dark, technical, and precise aesthetic that signals cutting-edge expertise.

### Style Palette
- **Backgrounds:** Deep Navy/Black (#0d1117)
- **Highlights:** Cyber Blue (#58a6ff) for links/interaction.
- **Status/Success:** Neon Green (#7ee787) for "Live" indicators and terminal outputs.
- **Typography:**
  - Headings: Monospace (e.g., JetBrains Mono) for a technical "agentic" feel.
  - Body: High-readability Sans-serif (e.g., Inter) for professional content.

### Recurring Elements
- **Glowing Borders:** Subtle neon hover effects on cards and buttons.
- **Agent Logs:** Boxed components styled as terminal output (`[STATUS] INITIALIZING...`) to highlight key wins.

---

## 3. Site Structure & Content
A focused, 4-page architecture optimized for speed and depth.

### Page Map
1.  **Home (The Command Center):**
    - High-impact hero: "Agentic Architect // E-commerce Ops."
    - **Live Status Terminal:** Real-time feed of GitHub activity and technical milestones.
    - **MCP Visualization:** Interactive diagram showing connections between AI tools, Shopify, and logistics.
2.  **Projects (The Engine):**
    - Deep dives into **OpenClaw** and **Hermes** harnesses.
    - Case studies on Shopify automation (n8n, custom scripts) at UHS Hardware.
    - Infrastructure projects (Proxmox, Coolify, Docker).
3.  **Research (The Lab):**
    - Blog-style entries on MCP server experiments, LLM benchmarks (Ollama), and Agentic workflows.
    - Focus on the "Personal Branding/Community" goal.
4.  **Stack (The Toolbox):**
    - Interactive breakdown of languages (Remix, TS, Python) and AI tools.

---

## 4. Professional Interactivity
Features that reinforce technical authority without becoming "gimmicky."

### Key Features
- **Command Palette (CMD + K):** Keyboard-driven navigation and search for power users.
- **MCP Node-Link Diagram:** An interactive SVG-based graph visualizing system architecture.
- **Live Activity Feed:** A streaming-style terminal component fetching data from the GitHub API and custom JSON sources.

---

## 5. Technical Architecture
Built for performance, scalability, and "infra-flexing."

- **Framework:** Remix (v2) with TypeScript.
- **Styling:** Vanilla CSS (Modular) for precise control and minimal bloat.
- **Data Layer:**
  - Content: MDX for research notes and project deep dives.
  - Real-time: Fetching from GitHub API and personal activity endpoints.
- **Infrastructure:**
  - Deployment: Self-hosted via **Coolify** on **Proxmox/Docker** to demonstrate full-stack infrastructure mastery.
  - Performance: Optimized for Core Web Vitals to align with SEO expertise.

---

## 6. Success Criteria
1.  The site loads in under 1 second (LCP).
2.  Interactive elements (CMD+K, Node Graph) work flawlessly on both Desktop and Mobile.
3.  The "Hybrid" positioning (Expert + Researcher) is clear to first-time visitors within 5 seconds of landing.
